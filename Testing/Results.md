# Результаты тестирования

## Пользовательский функционал

|Назначение|Сценарий|Ожидаемый результат|Фактический результат|Оценка|
|:---|:---|:---|:---|:---|
|Регистрация|1. Нажатие кнопки "Профиль". 2. Нажатие кнопки "Регистрация". 3. Ввод обязательных полей для регистрации.|Добавление пользователя в базу данных. |Соответствие ожидаемому результату.|Тест пройден|
|Авторизация.|1. Нажатие кнопки "Профиль". 2. Нажатие кнопки "Войти". 3. Ввод логина и пароля.|На сервере валидируется пароль и логин. Если введённые пользователем данные совпадают с данными из БД. Устанавливаются cookie и сессия для этого пользователя. Появляется доступ к корзине покупок.|Соответствие ожидаемому результату.|Тест пройден|
|Бронирование зала|1. Нажатие кнопки "Бронирование". 2. Выбор зала и нажатие кнопки "Забронировать". 3. Ввод обязательных полей. 4. Нажатие кнопки "Забронировать зал"|На сервере валидируется данные, если введено верно - запись в БД.|Соответствие ожидаемому результату.| Тест пройден|
|Просмотр информации о блюде|1. Нажатие кнопки "Меню". 2. Нажатие на название категории. 3. Нажатие на название блюда.|Открытие страницы с информацией о блюде|Соответствие ожидаемому результату.| Тест пройден|
|Добавление блюда в корзину|1. Нажатие кнопки "Меню". 2. Нажатие на название категории. 3. Нажатие на кнопку "Добавить блюдо в заказ".|Блюдо добавлено в корзину и в БД как заказ.|Соответствие ожидаемому результату.| Тест пройден|
|Оформление заказа|1. Добавить блюдо в корзину. 2. Перейти в корзину. 3. Ввод обязательных полей. 4. Нажатие на кнопку "Сделать заказ" .|Заказ добавлен к остальным вашим заказам и в БД|Соответствие ожидаемому результату.| Тест пройден|
|Комментарии к заказу|1. Перейти в заказы. 2. Нажатие на заказ. 3. Ввод комментария и оценки от 1 до 5. 4. Нажатие на кнопку "Оставить комментарий".|Комментарий добавлен в БД|Соответствие ожидаемому результату.| Тест пройден|
|Оставить отзыв|1. Переход на главную страницу при помощи кнопки "Домой" либо "Tahan Cafe". 2. В разделе "Свяжитесь с нами" ввести необходимые данные и отзыв. 3. Нажатие кнопки "Оставить сообщение".|Отзыв сохранен в БД|Соответствие ожидаемому результату.| Тест пройден|
|Отмена заказа|1. Переход на страницу заказов. 2. Нажатие на кнопку "отменить заказ".|Удаление заказа из БД|Соответствие ожидаемому результату.| Тест пройден|

|Обновление блюда|1. Нажатие кнопки "Меню". 2. Ввод новых данных под нужным блюдом. 3. Нажатие на кнопку "обновить"|Обновление данных по блюдам в БД. В меню отображается новая информацию.|Соответствие ожидаемому результату.| Тест пройден|
|Добавление нового блюда|1. Нажатие кнопки "Меню". 2. Ввод необходимых данных в свободную форму. 3. Нажатие на кнопку "добавить".|Добавление блюда в БД. Отображение нового блюда в меню.|Соответствие ожидаемому результату.| Тест пройден|
|Удаление категории|1. Нажатие кнопки "Категории". 2. Нажатие на картинку с нужной категорией.|Категория удаляется из БД. В меню категория больше не отображается.|Соответствие ожидаемому результату.| Тест пройден|
|Добавление категории|1. Нажатие кнопки "Категории". 2. Ввод необходимых данных в свободную форму. 3. Нажатие на кнопку "добавить".|Добавление категории в БД. Отображение новой категории в меню и в разделе категории.|Соответствие ожидаемому результату.| Тест пройден|
|Удаление ингредиентов|1. Нажатие кнопки "Ингредиенты". 2. Нажатие на картинку с ингредиентом. 3. Нажатие на кнопку "удалить".|Ингредиент удаляется из БД. Ингредиент больше не отображается в разделе ингредиентов.|Соответствие ожидаемому результату.| Тест пройден|
|Добавление ингредиентов|1. Нажатие кнопки "Ингредиенты". 2. Ввод необходимых данных в свободную форму. 3. Нажатие на кнопку "добавить".|Добавление ингредиента в БД. Отображение нового ингредиента в разделе ингредиенты.|Соответствие ожидаемому результату.| Тест пройден|
|Удаление зала|1. Нажатие кнопки "Залы". 2. Нажатие на кнопку "удалить" под нужным залом.|Зал удаляется из БД. Зал больше не отображается.|Соответствие ожидаемому результату.| Тест пройден|
|Добавление зала|1. Нажатие кнопки "Залы". 2. Ввод необходимых данных в свободную форму. 3. Нажатие на кнопку "добавить".|Добавление зала в БД. Отображение нового зала на странице.|Соответствие ожидаемому результату.| Тест пройден|
|Просмотр заказов|1. Нажатие кнопки "Заказы".|Открытие страницы с информацией о заказах|Соответствие ожидаемому результату.| Тест пройден|
|Просмотр зарегистрированных пользователей|1. Нажатие кнопки "Пользователи". 2. Нажатие на пользователя.|Открытие страницы с информацией о зарегистрированных пользователях. Вывод информации о конкретном пользователе.|Соответствие ожидаемому результату.| Тест пройден|
|Редактирование пользователей|1. Нажатие кнопки "Пользователи". 2. Нажатие на пользователя. 3. Ввод необходимых данных. 4. Нажатие кнопки "Сохранить изменения".|Вывод информации о конкретном пользователе. Обновление данных в БД.|Соответствие ожидаемому результату.| Тест пройден|
|Подтверждение заказа|1. Нажатие кнопки "Заказы". 2. Нажатие на необходимый заказ. 3. Ввод необходимых данных. 4. Нажатие на кнопку "Сохранить изменения".|Изменение информации о заказе в БД. Изменение статуса заказа у пользователя.|Соответствие ожидаемому результату.| Тест пройден|
